📖 Project Summary

AI Learning Buddy Maya is an AI-powered educational application developed using Python, Streamlit, and the OpenRouter API. The application helps beginners learn Python Programming Basics through simple explanations, real-life examples, interactive quizzes, and personalized guidance. Maya acts as a friendly and patient virtual tutor, making programming concepts easy to understand while encouraging self-paced learning.

✨ Features
📘 Beginner-friendly Python concept explanations
🌍 Real-life examples for better understanding
📝 Automatic quiz generation with answers
💬 Ask Anything feature for Python-related questions
🤖 Friendly AI tutor (Maya)
⚡ Fast and interactive Streamlit interface
📚 Personalized learning experience
